# 조코의 장-나물-밥

이 저장소는 한국의 전통 발효 장, 제철 나물, 밥 요리를 소개하는 정적 웹사이트입니다. 프로젝트 구조는 다음과 같습니다.

- `index.html`, `about.html`: 사이트의 메인 페이지와 소개 페이지
- `recipes/`: 장, 나물, 밥 카테고리별 인덱스와 세부 레시피 페이지 모음
  - `recipes/jang/`: 장 관련 레시피와 인덱스
  - `recipes/namul/`: 나물 레시피와 인덱스
  - `recipes/rice/`: 밥 요리 레시피와 인덱스
- `guides/`: 비건·할랄 가이드와 한국 식재료 설명 페이지
- `assets/css/style.css`: 공통 스타일 정의
- `assets/img/`: 로고, 레시피, 식재료 이미지를 위한 서브 디렉터리 (placeholder)

## 개발 메모

정적 HTML과 CSS만을 사용하므로 별도의 빌드 과정이 필요하지 않습니다. 레이아웃과 내비게이션을 통일하기 위해 모든 페이지는 `/assets/css/style.css`를 참조합니다. 이미지가 추가될 경우 해당 범주에 맞는 `assets/img/` 하위 디렉터리에 배치해 주세요.
